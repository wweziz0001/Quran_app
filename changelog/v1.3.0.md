# [1.3.0] - 2025-01-19

## ✨ الميزات الجديدة

### محرر SQL المتقدم (Monaco SQL Editor)

**الوصف:**
تم استبدال محرر النصوص العادي بمحرر Monaco Editor المتقدم مع دعم كامل لـ SQL.

**الميزات:**
- Syntax Highlighting لغة SQL
- Autocomplete لأسماء الجداول والأعمدة والكلمات المفتاحية
- Multiple Tabs للعمل على عدة استعلامات
- Query History مع عرض آخر 100 استعلام
- Saved Queries Library مع دعم المفضلة
- Query Formatting تلقائي
- اختصارات لوحة المفاتيح (Ctrl/Cmd + Enter للتنفيذ، Ctrl/Cmd + S للحفظ)
- Results Grid مع Sorting و Filtering
- Export Results بصيغ JSON, CSV, SQL

**الملفات المتأثرة:**
- `src/components/admin/database/sql-editor.tsx` - مكون جديد

---

### تصور المخطط (Schema Visualizer)

**الوصف:**
إضافة ER Diagram تفاعلي لعرض هيكل قاعدة البيانات مع العلاقات.

**الميزات:**
- React Flow للتصور التفاعلي
- عرض الجداول مع الأعمدة وأنواع البيانات
- عرض Primary Keys و Foreign Keys
- عرض العلاقات بين الجداول
- Zoom و Pan و Drag
- Auto Layout تلقائي
- تصدير كصورة

**الملفات المتأثرة:**
- `src/components/admin/database/schema-visualizer.tsx` - مكون جديد

---

### محرر البيانات (Data Editor)

**الوصف:**
إضافة واجهة CRUD كاملة لتحرير البيانات مباشرة من الواجهة.

**الميزات:**
- Inline Editing للخلايا
- Add Row مع نموذج إدخال
- Delete Row مع تأكيد
- Bulk Delete للصفوف المحددة
- Pagination متقدم
- Search و Filter
- Copy Row as JSON
- Export البيانات

**الملفات المتأثرة:**
- `src/components/admin/database/data-editor.tsx` - مكون جديد
- `src/app/api/admin/database/data/[table]/route.ts` - إضافة POST endpoint

---

### نظام الصلاحيات (Permissions System)

**الوصف:**
إضافة نظام صلاحيات متعدد المستويات للتحكم في الوصول.

**الأدوار:**
- **Viewer (مشاهد):** عرض الجداول والبيانات، تنفيذ SELECT فقط
- **Editor (محرر):** عرض، إضافة، تعديل، حذف، استيراد/تصدير، عرض سجلات التدقيق
- **Admin (مدير):** صلاحيات كاملة بما في ذلك النسخ الاحتياطي وتغيير الهيكل

**الملفات المتأثرة:**
- `src/lib/db-permissions.ts` - ملف جديد

---

### مراقبة الأداء (Performance Monitor)

**الوصف:**
إضافة لوحة مراقبة أداء قاعدة البيانات.

**الميزات:**
- Database Size و Table Count و Index Count
- Average Query Time و Query Count
- Slow Queries Detection (> 100ms)
- Table Statistics مع Row Count و Size
- Performance Tips
- Auto-refresh قابل للتخصيص

**الملفات المتأثرة:**
- `src/components/admin/database/performance-monitor.tsx` - مكون جديد
- `src/app/api/admin/db/metrics/route.ts` - تحسينات
