## 🎨 تحسينات التصميم

### تحسين شكل أرقام الآيات لتطابق تماماً أرقام السور (الحالة غير المختارة)

**المشكلة:**  
أرقام الآيات كانت تظهر بخلفية متدرجة داكنة (للحالة المختارة)، بينما في الفهرس أرقام السور غير المختارة تظهر بخلفية فاتحة وبسيطة.

**الحل:**  
تحديث تصميم أرقام الآيات لتكون مطابقة تماماً لتصميم أرقام السور غير المختارة في الفهرس.

**الملف:** `src/components/quran/islamic-ornaments.tsx`

---

## مقارنة التصميم

### قبل التحديث:
```
الوضع الفاتح: خلفية متدرجة (emerald-400 → teal-600) + نص أبيض
الوضع الداكن: خلفية متدرجة (amber-400 → amber-600) + نص داكن
الزاوية المزخرفة: bg-white/30 (للحالتين)
```

### بعد التحديث (مطابق للفهرس):
```
الوضع الفاتح: خلفية emerald-50 + نص emerald-700 + حدود emerald-200
الوضع الداكن: خلفية slate-700/80 + نص slate-300 + حدود amber-500/20
الزاوية المزخرفة: emerald-300/50 (فاتح) / amber-500/20 (داكن)
```

## التغييرات التفصيلية

```tsx
// قبل
<span className={cn(
  'relative w-11 h-11 rounded-xl flex items-center justify-center shrink-0',
  isDark 
    ? 'bg-gradient-to-br from-amber-400 via-amber-500 to-amber-600 text-slate-900'
    : 'bg-gradient-to-br from-emerald-400 via-emerald-500 to-teal-600 text-white'
)}>
  <span className="text-sm font-bold" style={{ fontFamily: "'Scheherazade New', 'Amiri', serif" }}>
    {toArabicNumber(number)}
  </span>
  <span className={cn(
    'absolute -top-1 -right-1 w-3 h-3 rotate-45',
    isDark ? 'bg-white/30' : 'bg-white/30'
  )} />
</span>

// بعد (مطابق للفهرس)
<span className={cn(
  'relative w-11 h-11 rounded-xl flex items-center justify-center shrink-0',
  isDark 
    ? 'bg-slate-700/80 text-slate-300 border border-amber-500/20'
    : 'bg-emerald-50 text-emerald-700 border border-emerald-200'
)}>
  <span className="text-sm font-bold">{toArabicNumber(number)}</span>
  <span className={cn(
    'absolute -top-1 -right-1 w-3 h-3 rotate-45',
    isDark ? 'bg-amber-500/20' : 'bg-emerald-300/50'
  )} />
</span>
```
