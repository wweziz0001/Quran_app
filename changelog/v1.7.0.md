## 🔧 التفاصيل التقنية

### Arabic Normalizer

```typescript
// تطبيع الحروف
'أ' → 'ا', 'إ' → 'ا', 'آ' → 'ا', 'ٱ' → 'ا'
'ة' → 'ه', 'ى' → 'ي', 'ؤ' → 'و'

// إزالة التشكيل
removeDiacritics('بِسْمِ اللَّهِ') → 'بسم الله'

// التطبيع للبحث
normalizeForSearch('بِسْمِ اللَّهِ الرَّحْمَٰنِ') → 'بسم الله الرحمن'
```

### Elasticsearch Configuration

```json
{
  "analyzer": {
    "arabic_analyzer": {
      "char_filter": ["arabic_normalizer", "diacritics_remover"],
      "tokenizer": "standard",
      "filter": ["lowercase", "arabic_stop", "arabic_stemmer", "arabic_synonyms"]
    }
  }
}
```

### Search Features

- **Text Search**: بحث نصي عادي
- **Fuzzy Search**: بحث تقريبي (fuzziness: AUTO)
- **Reference Search**: البحث بالمرجع (2:255)
- **Autocomplete**: اقتراحات فورية
- **Highlighting**: إبراز النتائج

## 🚀 أوامر التشغيل

```bash
# تشغيل Elasticsearch
cd elastic && docker-compose up -d

# انتظار جاهزية Elasticsearch
curl -X GET "localhost:9200/_cluster/health?wait_for_status=yellow"

# إنشاء الفهرس
bun run elastic/scripts/create-index.ts

# فهرسة جميع الآيات
bun run elastic/scripts/index-ayahs.ts

# اختبار البحث
bun run elastic/scripts/test-search.ts

# فتح Kibana
open http://localhost:5601
```
