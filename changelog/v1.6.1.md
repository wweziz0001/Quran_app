# [1.6.1] - 2025-01-17

## 🔧 الإصلاحات

### ربط تبويب Deployment بـ Microservices

**المشكلة:**
تبويب `/admin/deployment` كان يعرض بيانات محاكاة (simulated) بدلاً من البيانات الحقيقية من الخدمات المصغرة. لم يكن متصلاً فعلياً بالـ microservices التي تم إنشاؤها في v1.6.0.

**السبب:**
- دالة `checkServicesHealth` كانت تعيد بيانات عشوائية بدلاً من استدعاء الخدمات الحقيقية
- لم يكن هناك API endpoint للتحقق من حالة الخدمات

**الحل:**

1. **إنشاء API endpoint جديد:**
   - `src/app/api/admin/deployment/services-health/route.ts`
   - يتحقق من جميع الخدمات المصغرة (8 خدمات)
   - يستخدم `XTransformPort` للمرور عبر Gateway
   - يعيد معلومات مفصلة: uptime, version, responseTime, checks

2. **تحديث DeploymentSection:**
   - تحديث واجهة `ServiceStatus` لتشمل حقول جديدة
   - إضافة واجهة `HealthCheckResponse`
   - تحديث `checkServicesHealth` لاستدعاء API الحقيقي
   - إضافة ملخص الحالة (healthy/unhealthy/not_running)
   - عرض وقت آخر فحص
   - عرض وقت الاستجابة لكل خدمة

**الملفات المتأثرة:**
- `VERSION` - تحديث من 1.6.0 إلى 1.6.1
- `src/app/api/admin/deployment/services-health/route.ts` - 🟢 جديد
- `src/components/admin/deployment-section.tsx` - تحديث شامل

**التفاصيل:**

```diff
// قبل (محاكاة)
const newStatuses = await Promise.all(
  services.map(async (service) => {
    return {
      name: service.name,
      status: 'healthy' as const,
      uptime: Math.floor(Math.random() * 86400),
      port: service.port,
    };
  })
);

// بعد (API حقيقي)
const response = await fetch('/api/admin/deployment/services-health');
const data: HealthCheckResponse = await response.json();
setServiceStatuses(data.data.services);
```

**النتيجة:**
- ✅ الاتصال الفعلي بالخدمات المصغرة
- ✅ عرض الحالة الحقيقية لكل خدمة (healthy/unhealthy/not_running)
- ✅ عرض وقت الاستجابة (responseTime)
- ✅ ملخص سريع في أعلى قسم الحالة
- ✅ عرض وقت آخر فحص

## 📊 حالات الخدمات

| الحالة | الوصف | اللون |
|--------|-------|-------|
| `healthy` | الخدمة تعمل بشكل طبيعي | 🟢 أخضر |
| `unhealthy` | الخدمة تعمل لكن هناك مشكلة | 🟡 برتقالي |
| `not_running` | الخدمة غير مشغلة | 🔴 أحمر |
| `unknown` | تعذر التحقق من الحالة | ⚪ رمادي |
