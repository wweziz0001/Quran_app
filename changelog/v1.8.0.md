| **Embeddings** | تمثيل رقمي للنصوص (1536 بُعد) |

## 🔧 التفاصيل التقنية

### Z-AI Client

```typescript
// استخدام z-ai-web-dev-sdk
import { getZAIClient, chatCompletion, generateEmbedding } from '@/lib/z-ai-client';

// توليد Embedding
const embedding = await generateEmbedding('بسم الله');

// محادثة
const response = await chatCompletion({
  messages: [
    { role: 'system', content: 'أنت عالم في القرآن...' },
    { role: 'user', content: 'ما معنى هذه الآية؟' }
  ]
});
```

### Semantic Search

```typescript
import { semanticSearch, hybridSearch } from '@/services/ai/semantic-search';

// بحث دلالي
const results = await semanticSearch('آيات عن الرحمة', { limit: 10 });

// بحث هجين (نصي + دلالي)
const hybrid = await hybridSearch('الله', { limit: 20 });
```

### Question Answering

```typescript
import { answerQuestion, explainAyah } from '@/services/ai/question-answering';

// إجابة على سؤال
const answer = await answerQuestion('ما هو الإيمان؟');

// شرح آية
const explanation = await explainAyah(1); // الفاتحة الآية الأولى
```

## 📡 API Endpoints

### POST /api/ai/search

```json
{
  "query": "آيات عن الرحمة",
  "type": "semantic",
  "limit": 10,
  "threshold": 0.5
}
```

### POST /api/ai/chat

```json
{
  "type": "qa",
  "question": "ما هو الإيمان؟",
  "options": { "language": "ar" }
}
```

### POST /api/ai/tafsir

```json
{
  "action": "explain",
  "ayahId": 1,
  "options": { "includeVocabulary": true }
}
```

### POST /api/ai/embeddings

```json
{
  "action": "generate_all"
}
```

### GET /api/ai/recommendations?ayahId=1
