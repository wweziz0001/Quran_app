## 📝 التغييرات

### `src/services/arabic-normalizer.ts`

```diff
- '\u0671', // Alef wasla - كان يُحذف
+ // Note: '\u0671' (Alef wasla - ٱ) is NOT removed here because it's normalized to 'ا' in CHAR_NORMALIZATIONS
```

**النتيجة:**
```typescript
// قبل الإصلاح
removeDiacritics('ٱللَّهِ') → 'لله'  // ❌ حُذفت الألف

// بعد الإصلاح
removeDiacritics('ٱللَّهِ') → 'ٱلله'  // ✅
normalizeForSearch('ٱللَّهِ') → 'الله'  // ✅
```
