# v1.1.9 - File Upload Feature

## New Features

### File Upload in Files Section
Added a complete file upload system to the admin panel's Files section:

#### Features:
- **Upload Button**: New "Upload" button in the top toolbar for quick access
- **Drag & Drop Support**: Drag files directly onto the drop zone
- **Multiple Files**: Upload multiple files at once
- **Custom Destination**: Specify the target folder for uploads (defaults to `public/uploads`)
- **Auto-create Folders**: Target folders are automatically created if they don't exist
- **File Size Display**: Shows file size for each selected file
- **Progress Tracking**: Visual feedback for each file upload (success/failure)
- **File Management**: Remove files from the upload queue before uploading

#### Technical Details:
- **Max File Size**: 10MB per file
- **Allowed Directories**: All directories that are already accessible in the file editor
- **API Endpoint**: Extended `POST /api/files` to handle `multipart/form-data` uploads
- **Security**: Path validation and security checks are applied to all uploads

#### UI Components:
- Upload dialog with folder path input
- Drag & drop zone with visual feedback
- File list with remove buttons
- Progress indicators (success/failure)
- Upload button with file count

## API Changes

### POST /api/files
Extended to support file uploads via `multipart/form-data`:

**Request (multipart/form-data):**
- `file`: The file to upload
- `path`: Target folder path

**Response:**
```json
{
  "success": true,
  "message": "File uploaded successfully",
  "data": {
    "name": "filename.ext",
    "path": "public/uploads/filename.ext",
    "size": 12345,
    "type": "file/type"
  }
}
```
