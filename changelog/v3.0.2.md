# [3.0.2] - 2026-03-02

## 🔧 الإصلاحات

### تحديث سكربت التشغيل التلقائي (setup_and_run.sh)

**المشكلة:**
سكربت `setup_and_run.sh` كان يستخدم مسارات نسبية ومعقدة، والمستخدم طلب تبسيطه.

**الحل:**
تحديث السكربت لاستخدام التنقل المباشر بين المجلدات `cd ../../services/...`.

**الملفات المتأثرة:**
- `setup_and_run.sh` - تبسيط التنقل بين الخدمات

**التفاصيل:**
```bash
# قبل (مسارات معقدة)
cd "$SERVICES_DIR/quran-service"

# بعد (تنقل مباشر)
cd services/quran-service
bun install
nohup bun run dev > output.log 2>&1 &

cd ../../services/audio-service
bun install
nohup bun run dev > output.log 2>&1 &
# ... وهكذا
```

**النتيجة:**
- ✅ سكربت أبسط وأوضح
- ✅ تشغيل جميع الخدمات الـ 8 + التطبيق الرئيسي
