## 🎨 تحسينات التصميم

### تحسين شكل أرقام الآيات لتطابق تماماً أرقام السور (مع حالة الاختيار)

**المشكلة:**  
1. نوع الخط في رقم الآية مختلف عن نوع الخط في رقم السورة
2. لا يتغير لون رقم الآية المختارة مثل السورة المختارة في الفهرس

**الحل:**  
تم تحديث تصميم أرقام الآيات لتكون مطابقة تماماً لتصميم أرقام السور في الفهرس.

---

## التغييرات التفصيلية

### 1. إضافة prop جديد `isSelected`

```tsx
// قبل
export const LuxuriousVerseNumber = ({ 
  number, 
  isDark 
}: { 
  number: number; 
  isDark: boolean;
}) => { ... }

// بعد
export const LuxuriousVerseNumber = ({ 
  number, 
  isDark,
  isSelected = false 
}: { 
  number: number; 
  isDark: boolean;
  isSelected?: boolean;
}) => { ... }
```

### 2. تحديث المنطق ليطابق الفهرس تماماً

```tsx
<span className={cn(
  'relative w-11 h-11 rounded-xl flex items-center justify-center shrink-0',
  'transition-all duration-300 group-hover:scale-105',
  isSelected 
    ? isDark
      ? 'bg-gradient-to-br from-amber-400 to-amber-600 text-slate-900 shadow-lg shadow-amber-500/30'
      : 'bg-gradient-to-br from-emerald-400 to-teal-600 text-white shadow-lg shadow-emerald-500/30'
    : isDark
      ? 'bg-slate-700/80 text-slate-300 border border-amber-500/20'
      : 'bg-emerald-50 text-emerald-700 border border-emerald-200'
)}
style={{
  boxShadow: isSelected
    ? isDark
      ? '0 0 20px rgba(217, 119, 6, 0.3)'
      : '0 0 20px rgba(5, 150, 105, 0.2)'
    : 'none'
}}
>
  <span className="text-sm font-bold">{toArabicNumber(number)}</span>
  <span className={cn(
    'absolute -top-1 -right-1 w-3 h-3 rotate-45',
    isSelected 
      ? 'bg-white/30' 
      : isDark 
        ? 'bg-amber-500/20' 
        : 'bg-emerald-300/50'
  )} />
</span>
```

### 3. تحديث استدعاء المكون

```tsx
// قبل
<LuxuriousVerseNumber number={verse.numberInSurah} isDark={isDarkMode} />

// بعد
<LuxuriousVerseNumber 
  number={verse.numberInSurah} 
  isDark={isDarkMode}
  isSelected={currentVerse?.id === verse.id}
/>
```

## المقارنة مع الفهرس

| الميزة | رقم السورة (الفهرس) | رقم الآية (بعد التحديث) |
|--------|---------------------|------------------------|
| **نوع الخط** | `text-sm font-bold` | ✅ `text-sm font-bold` |
| **غير مختار (فاتح)** | `bg-emerald-50 text-emerald-700 border-emerald-200` | ✅ مطابق |
| **غير مختار (داكن)** | `bg-slate-700/80 text-slate-300 border-amber-500/20` | ✅ مطابق |
| **مختار (فاتح)** | `bg-gradient-to-br from-emerald-400 to-teal-600 text-white` | ✅ مطابق |
| **مختار (داكن)** | `bg-gradient-to-br from-amber-400 to-amber-600 text-slate-900` | ✅ مطابق |
| **الزاوية المزخرفة** | نفس المنطق | ✅ مطابق |
| **التوهج عند الاختيار** | `shadow-lg shadow-emerald-500/30` | ✅ مطابق |
