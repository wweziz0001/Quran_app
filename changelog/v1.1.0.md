## ✨ الميزات المضافة

### 1. محرر SQL في مستعرض قاعدة البيانات

**الوصف:**  
تم إضافة تبويب جديد "محرر SQL" في قسم Database يتيح تنفيذ استعلامات SQL مباشرة على قاعدة البيانات.

**المميزات:**
- تنفيذ استعلامات SELECT, PRAGMA, EXPLAIN فقط (للأمان)
- 8 استعلامات جاهزة للاستخدام السريع
- سجل آخر 20 استعلام مع حالة النجاح/الفشل
- عرض النتائج في جدول منسق مع الترقيم
- اختصار Ctrl+Enter للتنفيذ السريع
- عرض وقت التنفيذ وعدد الصفوف

**الملفات الجديدة:**
```
src/app/api/admin/database/sql/route.ts  (جديد)
```

**الملفات المعدلة:**
```
src/components/admin/database-browser.tsx
```

**التغييرات الرئيسية:**
