### 1. أخطاء أسماء العلاقات في Prisma

**الوصف:**  
حدثت أخطاء `Unknown field for select statement` في عدة API endpoints بسبب استخدام أسماء خاطئة للعلاقات عند استخدام `_count` في Prisma.

**المشكلة:**
في Prisma، عند استخدام `_count` مع `select`، يجب استخدام أسماء العلاقات كما هي معرفة في الـ schema (PascalCase)، وليس أسماء مخصصة.

**الملفات المصححة:**
```
src/app/api/tafsir/route.ts
src/app/api/admin/image-mushaf/route.ts
src/app/api/admin/ttf-mushaf/route.ts
```

**التغييرات:**

##### tafsir/route.ts
```diff
  const sources = await db.tafsirSource.findMany({
    where,
    orderBy: [
      { isDefault: 'desc' },
      { nameEnglish: 'asc' },
    ],
    include: {
      _count: {
-       select: { tafsirEntries: true },
+       select: { TafsirEntry: true },
      },
    },
  });

  // ...
- entriesCount: s._count.tafsirEntries,
+ entriesCount: s._count.TafsirEntry,
```

##### image-mushaf/route.ts
```diff

  // ...

```

### 2. عدم ظهور الآيات في إدارة القرآن

**الوصف:**  
عند استعراض آيات أي سورة من قسم "إدارة القرآن"، لا تظهر الآيات بالرغم من وجودها في قاعدة البيانات.

**السبب:**  
عدم وجود حالة تحميل (`loading state`) للآيات، مما يجعل الـ Dialog يعرض "لا توجد آيات" قبل اكتمال عملية جلب البيانات من الـ API.

**الملف:** `src/components/admin/quran-management-section.tsx`
