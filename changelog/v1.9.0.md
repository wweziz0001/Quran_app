## ✨ الميزات الجديدة

### نظام بث الصوت الاحترافي (Professional Audio Streaming)

تم إنشاء نظام متكامل لبث الصوت يشمل:

| الميزة | الوصف |
|--------|-------|
| **HLS Streaming** | بث HTTP Live Streaming بجودات متعددة |
| **Adaptive Bitrate** | 64k, 128k, 192k, 256k |
| **Pre-signed URLs** | روابط آمنة موقعة للبث |
| **Redis Caching** | تخزين مؤقت للروابط والمetadata |
| **FFmpeg Integration** | تحويل ومعالجة الصوت |

## 📂 الملفات الجديدة

### Services (`src/services/audio/`)
- `hls-converter.ts` - تحويل الصوت إلى HLS (~400 سطر)
- `storage.ts` - التخزين المحلي/S3 (~300 سطر)
- `cache.ts` - Redis caching (~250 سطر)
- `audio-processor.ts` - معالجة الصوت (~350 سطر)
- `pre-signed.ts` - روابط موقعة (~200 سطر)
- `index.ts` - نقطة الدخول الرئيسية

### API Endpoints (`src/app/api/audio/`)
- `stream/route.ts` - GET بث الصوت
- `upload/route.ts` - POST رفع الملفات
- `process/route.ts` - POST/GET معالجة الصوت
- `status/route.ts` - GET/DELETE حالة المعالجة

### Mini Service (`mini-services/audio-processor/`)
- `index.ts` - خدمة المعالجة الخلفية
- `package.json` - إعدادات الحزمة

### Scripts (`scripts/`)
- `convert-to-hls.ts` - تحويل الصوت إلى HLS
- `sync-audio-files.ts` - مزامنة الملفات من API خارجي

## 🔧 التفاصيل التقنية

### HLS Converter
```typescript
// تحويل ملف صوتي إلى HLS
const result = await convertToHLS({
  inputPath: './audio.mp3',
  outputDir: './hls/',
  segmentDuration: 10,
  bitrates: [64, 128, 192],
});

// النتيجة
{
  playlistUrl: './hls/master.m3u8',
  segments: ['./hls/64k/segment_001.ts', ...],
  duration: 45.2,
  bitrates: [64, 128, 192]
}
```

### Storage Service
```typescript
// رفع ملف
const result = await uploadFile('./audio.mp3', 'audio/hls/1/1.mp3');

// الحصول على رابط موقّع
const url = await getPresignedUrl('audio/hls/1/1.mp3', 3600);
```

### Cache Service
```typescript
// تخزين مؤقت
await cacheUrl('1:1:hls', streamUrl, 3600);

// استرداد من الذاكرة المؤقتة
const cached = await getCachedUrl('1:1:hls');

// نمط Get or Set
const url = await getOrSet('key', async () => {
  return await generateUrl();
});
```

### Pre-signed URLs
```typescript
// إنشاء رابط موقّع
const url = generatePresignedUrl({
  key: 'audio/hls/1/1.mp3',
  expiresIn: 3600,
  ip: '192.168.1.1',
});

// التحقق من الرابط
const { valid, key } = validatePresignedUrl(token, ip);
```

## 🌐 API Endpoints

### GET /api/audio/stream
```json
{
  "success": true,
  "data": {
    "type": "hls",
    "playlistUrl": "/storage/audio/hls/1/1/master.m3u8?token=...",
    "duration": 45200,
    "bitrate": 128
  }
}
```

### POST /api/audio/upload
```json
// Request
{
  "file": "<binary>",
  "recitationId": "ar.alafasy",
  "ayahId": 1,
  "processNow": true
}

// Response
{
  "success": true,
  "data": {
    "id": "ar.alafasy_1_1234567890",
    "status": "processing"
  }
}
```

### GET /api/audio/status
```json
{
  "success": true,
  "data": {
    "audioFiles": {
      "total": 6236,
      "byStatus": { "ready": 6000, "processing": 236 },
      "byFormat": [{ "format": "hls", "count": 6000, "totalSize": 524288000 }],
      "totalSize": 524288000
    },
    "services": {
      "ffmpeg": "available",
      "cache": "active"
    }
  }
}
```
