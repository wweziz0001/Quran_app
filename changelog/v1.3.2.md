# [1.3.2] - 2025-03-01

## ✨ الميزات الجديدة

### Schema Visualizer - تحسينات العرض والعلاقات

**المشكلة:**
- لم تظهر العلاقات بين الجداول في Schema Tab
- لم يكن هناك تفاعل عند التأشير على خطوط العلاقات
- الجداول متقاربة جداً مما يصعب قراءة العلاقات

**السبب:**
1. SQLite لا يخزن Foreign Keys على مستوى قاعدة البيانات (Prisma يديرها على مستوى التطبيق)
2. لم يكن هناك API لاستخراج العلاقات من Prisma Schema
3. المسافة بين الجداول كانت صغيرة (50px)

**الحل:**

#### 1. إضافة API لاستخراج العلاقات من Prisma Schema
```typescript
// src/app/api/admin/db/schema/route.ts
// يستخرج العلاقات من prisma/schema.prisma
// يعيد 23 علاقة بين الجداول
```

#### 2. إضافة إبراز تفاعلي للعلاقات
عند التأشير على خط العلاقة:
- ✨ يتوهج الجدولان المتصلان بإطار أزرق
- ✨ يتم إبراز الحقلين المتصلين بخلفية زرقاء
- ✨ يصبح الخط أكثر سماكة وتوهجاً

#### 3. زيادة المسافة بين الجداول
- من `50px` إلى `150px` (ثلاثة أضعاف)
