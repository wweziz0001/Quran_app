# [1.5.1] - 2025-01-16

## ✨ الميزات الجديدة - استكمال Stage 1

### إكمال الملفات الناقصة لـ Stage 1

بعد إضافة الجداول الجديدة في v1.5.0، تم إنشاء الملفات الناقصة:

---

### 1. ملف البذرة (prisma/seed.ts)

ملف كامل لتعبئة البيانات الأولية:

- **7 قواعد تجويد**: إدغام، إظهار، إقلاب، إخفاء، غنة، مد، قلقلة
- **5 ميزات (Feature Flags)**: semantic_search, audio_hls, offline_mode, dark_mode, word_analysis
- **6 إعدادات تطبيق**: default_reciter, default_tafsir, default_translation, etc.
- **114 slug للسور**: جميع سور القرآن مع slugs صديقة للـ SEO

**الاستخدام:**
```bash
bun run db:seed
```

---

### 2. أنواع TypeScript (src/types/database.ts)

أنواع كاملة للجداول الجديدة:

| النوع | الوصف |
|-------|--------|
| `SearchIndex` | فهرس البحث الدلالي |
| `WordAnalysis` | تحليل الكلمات صرفياً |
| `Notification` | إشعارات المستخدم |
| `Analytics` | تحليلات الاستخدام |
| `AudioFile` | ملفات الصوت HLS |
| `Collection` | مجموعات المستخدم |
| `TajweedRule` | قواعد التجويد |
| `FeatureFlag` | ميزات التطبيق |
| `APILog` | سجل طلبات API |
| `UserSession` | جلسات المستخدم |

**أنواع إضافية:**
- `ApiResponse<T>` - استجابة API موحدة
- `PaginatedResponse<T>` - استجابة مع صفحات
- `CreateInput<T>` - نوع للإنشاء
- `UpdateInput<T>` - نوع للتحديث

**ثوابت:**
- `FEATURE_FLAG_KEYS` - مفاتيح الميزات
- `TAJWEED_CODES` - رموز قواعد التجويد

---

### 3. API Endpoints الجديدة

#### `/api/search-index`
| Method | الوصف |
|--------|--------|
| GET | جلب فهارس البحث (بآية أو بلغة) |
| POST | إنشاء/تحديث فهرس بحث |
| DELETE | حذف فهرس بحث |

#### `/api/notifications`
| Method | الوصف |
|--------|--------|
| GET | جلب إشعارات المستخدم |
| POST | إنشاء إشعار جديد |
| PATCH | تحديد كمقروء (أو تحديد الكل) |
| DELETE | حذف إشعار |

#### `/api/analytics`
| Method | الوصف |
|--------|--------|
| GET | جلب ملخص التحليلات |
| POST | تسجيل حدث جديد |
| DELETE | حذف التحليلات القديمة |

#### `/api/feature-flags`
| Method | الوصف |
|--------|--------|
| GET | جلب الميزات (الكل أو بواسطة key) |
| POST | إنشاء ميزة جديدة |
| PATCH | تحديث ميزة |
| DELETE | حذف ميزة |

#### `/api/tajweed-rules`
| Method | الوصف |
|--------|--------|
| GET | جلب القواعد (الكل أو بواسطة code) |
| POST | إنشاء قاعدة جديدة |
| PATCH | تحديث قاعدة |
| DELETE | حذف قاعدة |
