# [1.5.0] - 2025-01-16

## ✨ الميزات الجديدة - Stage 1: Enterprise Database Schema

### 1. إضافة 11 جدول جديد إلى قاعدة البيانات

#### 1.1 SearchIndex (فهرس البحث الدلالي)
```prisma
model SearchIndex {
  id               String   @id
  ayahId           Int      @unique
  content          String
  contentNormalized String?
  embedding        String?  // JSON vector للبحث الدلالي
  embeddingModel   String?
  language         String   @default("ar")
}
```
- **الهدف**: دعم البحث الدلالي باستخدام AI
- **الاستخدام**: تخزين embeddings للآيات

#### 1.2 WordAnalysis (تحليل الكلمات)
```prisma
model WordAnalysis {
  id             Int
  ayahId         Int
  wordPosition   Int
  word           String
  root           String?  // الجذر
  lemma          String?  // الليما
  pos            String?  // Part of Speech
  morphology     String?  // JSON تفاصيل الصرف
}
```
- **الهدف**: تحليل صرفي للكلمات
- **الاستخدام**: البحث بالجذر، التحليل النحوي

#### 1.3 Notification (الإشعارات)
```prisma
model Notification {
  id        String
  userId    String
  type      String  // BOOKMARK, READING_REMINDER, etc.
  title     String
  message   String
  read      Boolean @default(false)
}
```
- **الهدف**: نظام إشعارات للمستخدمين
- **الاستخدام**: تذكير بالقراءة، تنبيهات

#### 1.4 Analytics (تحليلات الاستخدام)
```prisma
model Analytics {
  id         String
  eventType  String  // PAGE_VIEW, AYAH_READ, etc.
  userId     String?
  surahId    Int?
  ayahId     Int?
  timestamp  DateTime
}
```
- **الهدف**: تتبع استخدام التطبيق
- **الاستخدام**: إحصائيات، تحسين التجربة

#### 1.5 UserSession (جلسات المستخدم)
```prisma
model UserSession {
  id           String
  userId       String
  token        String   @unique
  refreshToken String?  @unique
  expiresAt    DateTime
}
```
- **الهدف**: إدارة جلسات المستخدمين
- **الاستخدام**: أمان، تسجيل دخول متعدد

#### 1.6 TajweedRule (قواعد التجويد)
```prisma
model TajweedRule {
  id          String
  nameArabic  String
  nameEnglish String
  code        String   @unique
  color       String?
  category    String?
}
```
- **الهدف**: تعريف قواعد التجويد
- **البيانات الأولية**: 7 قواعد (إدغام، إظهار، إقلاب، إخفاء، غنة، مد، قلقلة)

#### 1.7 AudioFile (ملفات الصوت HLS)
```prisma
model AudioFile {
  id              String
  recitationId    String
  ayahId          Int
  format          String
  hlsPlaylistUrl  String?
  status          String  // pending, processing, ready
}
```
- **الهدف**: دعم بث الصوت HLS
- **الاستخدام**: جودة متعددة، بث سلس

#### 1.8 Collection & CollectionItem (مجموعات المستخدم)
```prisma
model Collection {
  id          String
  userId      String
  name        String
  isPublic    Boolean
  itemCount   Int
}

model CollectionItem {
  id           String
  collectionId String
  ayahId       Int
  note         String?
}
```
- **الهدف**: مجموعات مخصصة للآيات
- **الاستخدام**: آيات التداوي، آيات الرزق، إلخ

#### 1.9 APILog (سجل طلبات API)
```prisma
model APILog {
  id           String
  endpoint     String
  method       String
  statusCode   Int
  responseTime Int
  timestamp    DateTime
}
```
- **الهدف**: مراقبة الأداء والأمان
- **الاستخدام**: rate limiting، تحليل الأخطاء

#### 1.10 FeatureFlag (ميزات التطبيق)
```prisma
model FeatureFlag {
  id          String
  key         String   @unique
  enabled     Boolean
  rollout     Int      // نسبة التفعيل 0-100
}
```
- **الهدف**: تفعيل/تعطيل الميزات بشكل ديناميكي
- **البيانات الأولية**: 5 ميزات (semantic_search, audio_hls, offline_mode, dark_mode, word_analysis)
