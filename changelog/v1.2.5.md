### 1. تحديث Prisma Schema
```prisma
model AuditLog {
  id           String   @id @default(cuid())
  timestamp    DateTime @default(now())
  action       String
  resourceType String
  resourceName String
  query        String?
  status       String
  // ... المزيد
}

model SavedQuery {
  id          String   @id @default(cuid())
  name        String
  query       String
  isFavorite  Boolean  @default(false)
  useCount    Int      @default(0)
  // ... المزيد
}

model BackupRecord {
  id        String   @id @default(cuid())
  name      String
  type      String
  size      Int
  status    String
  checksum  String?
  // ... المزيد
}

model QueryHistory {
  id             String   @id @default(cuid())
  query          String
  executionTime  Float
  status         String
  // ... المزيد
}
```

### 3. API Endpoints

| Endpoint | Method | الوصف |
|----------|--------|-------|
| `/api/admin/db/tables` | GET | قائمة الجداول مع الهيكل |
| `/api/admin/db/query` | POST | تنفيذ استعلامات SQL |
| `/api/admin/db/metrics` | GET | إحصائيات قاعدة البيانات |
| `/api/admin/db/health` | GET | فحص الاتصال |
| `/api/admin/db/backup` | GET/POST | إدارة النسخ الاحتياطي |
| `/api/admin/db/audit-logs` | GET | سجلات التدقيق |
| `/api/admin/db/saved-queries` | GET/POST | الاستعلامات المحفوظة |

- **ImportExportTab**: استيراد وتصدير البيانات
- **BackupTab**: إدارة النسخ الاحتياطي
