## 🔧 الإصلاحات

### إصلاح الوصول لملفات changelog و VERSION في محرر الملفات

**المشكلة:**  
لم يكن بالإمكان استعراض ملفات مجلد `changelog/` وملف `VERSION` في محرر الملفات (Files) بلوحة التحكم، حيث كان يظهر خطأ 403 Access Denied.

**السبب:**  
قيود الأمان في API لم تكن تتضمن مجلد `changelog/` وملف `VERSION` في قائمة الملفات المسموح بالوصول إليها.

**الملف:** `src/app/api/files/route.ts`

**التغييرات:**

```diff
  // Allowed directories for security
  const ALLOWED_PATTERNS = [
    /^src\//,           // src/**/*
    /^prisma\//,        // prisma/**/*
    /^docs\//,          // docs/**/*
    /^public\//,        // public/**/*
+   /^changelog\//,     // changelog/**/*
    /^components\.json$/,
    /^package\.json$/,
    /^tsconfig\.json$/,
    /^tailwind\.config\.ts$/,
    /^next\.config\.ts$/,
    /^\.env\.example$/,
    /^bun\.lockb$/,
    /^postcss\.config\.mjs$/,
+   /^VERSION$/,        // VERSION file
  ];
```
