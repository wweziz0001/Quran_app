# v1.2.6 - 2026-02-28

## 🔧 الإصلاحات

### إصلاح #1 - عدم عمل واجهة مدير قواعد البيانات (DB Manager)

**المشكلة:**
واجهة مدير قواعد البيانات لم تكن تعمل بشكل صحيح، حيث كانت جميع التبويبات لا تعرض أي بيانات:
- Dashboard: لا يعرض أي بيانات
- Tables: لا يظهر فيها أي بيانات
- Audit Logs: لا يظهر فيها أي بيانات
- Backup: الأزرار لا تعمل ولا تقوم بأي شيء
- Import/Export: لا تظهر فيها أي بيانات ولا تعمل فيها الأزرار

**الأسباب:**
1. عدم وجود ملف `.env` مع `DATABASE_URL` المطلوب للاتصال بقاعدة البيانات
2. خطأ `TypeError: Do not know how to serialize a BigInt` عند تحويل بيانات SQLite إلى JSON
3. استعلام `dbstat` غير مدعوم في SQLite الافتراضي
4. زر Export لم يكن لديه دالة معالجة (handler function)

**الملفات المتأثرة:**

#### 1. إنشاء ملف `.env` (جديد)
```env
# Database
DATABASE_URL="file:./db/custom.db"

# NextAuth
NEXTAUTH_SECRET="your-super-secret-key-change-in-production"
NEXTAUTH_URL="http://localhost:3000"

# App Settings
NODE_ENV="development"
```

#### 5. إضافة وظيفة Export في `src/components/admin/database/database-manager.tsx`
```typescript
const handleExport = async () => {
  if (!exportTable) { toast.error('Please select a table to export'); return; }
  setProcessing(true);
  try {
    const response = await fetch(`/api/admin/database/data/${exportTable}?limit=10000`);
    const result = await response.json();
    if (result.success) {
      let content: string;
      let filename: string;
      let mimeType: string;

      if (exportFormat === 'json') {
        content = JSON.stringify(result.data, null, 2);
        filename = `${exportTable}.json`;
        mimeType = 'application/json';
      } else if (exportFormat === 'csv') {
        // CSV generation...
      } else {
        // SQL generation...
      }

      const blob = new Blob([content], { type: mimeType });
      const url = URL.createObjectURL(blob);
      const a = document.createElement('a');
      a.href = url;
      a.download = filename;
      a.click();
      URL.revokeObjectURL(url);
      toast.success(`Exported ${result.data.length} rows`);
    }
  } catch { toast.error('Export failed'); }
  finally { setProcessing(false); }
};
```
